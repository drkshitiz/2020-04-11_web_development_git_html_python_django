## HTML

```html
<!DOCTYPE html>
<html>
<head>
  <title>My Web Page!</title>
</head>
<body>
Hello, world!
<h1>Heading-1</h1>
<h2>Heading-2</h2>
<h3>Heading-3</h3>
<h4>Heading-4</h4>
<h5>Heading-5</h5>
<h6>Heading-6</h6>
An ordered list:
<ol>
  <li>item-1</li>
  <li>item-2</li>
  <li>item-3</li>
</ol>
An unordered list:
<ul>
  <li>item-1</li>
  <li>item-2</li>
  <li>item-3</li>
</ul>
Add image:
<img src = "dog.jpg" height = "200" width = "300">
[If we want to maintain the aspect ratio of the image, add only one attribute, either height or width.]
<img src="dog.jpg" width="50%"">


<form>
<input type="text" placeholder="Full Name" name="name">
<button>Submit!</button>
</form>
</body>
</html>
```
